# Status Code

Status Code는 서버가 요청 결과를 알려주는 번호이다.

## 200 OK

요청이 정상적으로 처리되었다.

주로

- GET
- PUT
- DELETE

에서 사용한다.

---

## 201 Created

새로운 데이터가 생성되었다.

주로

- POST

에서 사용한다.

---

## 404 Not Found

요청한 데이터를 찾을 수 없을 때 사용한다.

예시

GET /users/100

---

## 400 Bad Request

잘못된 요청을 보냈을 때 사용한다.

예시

필수 데이터 누락
잘못된 형식의 데이터

---

## 기억하기

200 → 조회, 수정, 삭제 성공

201 → 생성 성공

404 → 찾을 수 없음

400 → 잘못된 요청
